Lab 8: Express Basics

Your Express server is running! Test these endpoints:

GET /api/info - Server information
GET /api/greet/:name - Personalized greeting
GET /api/search?q=term - Search with query string
POST /api/echo - Echo posted data
GET /api/messages - Get all messages
POST /api/messages - Create a message
GET /api/users - Get all users

Testing

Run npm test to check your implementation.

Use Postman or curl to test the endpoints manually.
